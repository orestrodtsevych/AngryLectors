Main menu

Game starts from main menu. Here we have 3 buttons:
"START" - start the game
"Settings" - go to game setting to change master volume, music volume and sound volume.
"Shop" - you can buy new character models and a lot of usefull stuff.

Start of the game

According to the plot, the character of the player did not close all the deadlines. The lecturer noticed this, he, as a good teacher, wants the student to pass everything as soon as possible, so the chase begins in the corridors of the university.

Gameplay

The main goal of the game is to make the player-controlled character run as far as possible, avoiding oncoming obstacles. His success is expressed in points, over time, running accelerates, and as a result of increasing speed increases the likelihood of encountering obstacles. If a character encounters something, the lecturer catches up with him and grabs (no matter how much he manages to escape), and the game ends.

The end

In the event of a collision with the teacher's desk or chair, the character falls to the floor and as a result the teacher catches the student, which means the end of the game. The teacher is satisfied with this development events. As a result, the student must close all deadlines.

At this stage we can go to the menu or start running away from the lecturer again
